[Wazo-plugin-my-own-dialplan](https://github.com/sboily/wazo-plugin-my-own-dialplan)

```bash
apt install wazo-plugind-cli
wazo-plugind-cli -c "install git https://github.com/sboily/wazo-plugin-my-own-dialplan --ref main"
```
